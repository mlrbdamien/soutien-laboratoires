Basé sur PR-6387
▶ Contrôles avant enregistrement — rappel rapide
  • Vérifier identité tubes = feuille de demande (y.c. micro-tubes)
  • Lecteur optique : nb codes enregistrés = nb coches sur la feuille
  • Prescription connectée : nb tubes reçus = nb notés/écran
  • Étiquettes bien collées, sans pli, bon tube, bon sens, bon état
  • NLab de l'étiquette = NLab de la feuille
  • Annotations manuscrites → Données cliniques > Libellé
  • Texte illisible → transmettre la feuille au labo concerné
  • Visa HAUT feuille = étiquetage · Visa BAS feuille = enregistrement

Tubes mères

Étiquettes commençant par un chiffre (1, 4, 5, 6, 7, 8...)

Étiquettes secondaires

Commencent par une lettre = labo destinataire :
H Hémato · C Chimie · IS Immuno · MM Microbio mol. · G Génétique

⚠ Libellé avec « ! »

Avertir le laboratoire concerné oralement ou par téléphone immédiatement.

Confidentialité

Étiquettes surnuméraires + ruban encreur → poubelles jaunes
Fiche pratique FP-7679 disponible à chaque poste.

✓ Automatique

Tubes conformes pour le 8100 sans pré-analytique particulière → checkés automatiquement par le 8100.

✋ Manuel

Tous les autres tubes doivent être checkés manuellement.

❄ Tubes congelés IS / MM / Tox

Sortir du sagex → congélateur près poste envois sur statif correspondant → transmettre étiquettes au labo concerné.

❄ Tubes congelés Chimie / Hémato

Laisser dans le sagex → déposer avec étiquettes sur le statif du labo concerné.

Garder tous les tubes sérum

(Neurologie, Hôpital de jour, etc.)
Code : SREST
Coller étiquettes « 1 » sur les tubes de sérum, donner la feuille + étiquettes « SREST » + tubes au poste 8100.

🔬 Chimie

  • Tubes conformes → 8100 (sauf : pas à centrifuger, pré-analytique spéciale, étiquette secondaire, pas assez de matériel)
  • Non conformes → statif bleu
  • Micro-tubes : étiquette « 1 » pour décantage (ne PAS coller sur micro-tube), pochette côté statif bleu, avertir poste 8100
  • Thermos (cryoglobuline, agglutinine froide) → côté statif bleu + avertir TAB chimie

🩸 Hématologie

  • EDTA sans étiquette secondaire et gris foncés → rouleau
  • Tous les autres → statif rouge
  • Routine matin : EDTA 2.6ml → racks A du DXH

🧫 Immuno-sérologie

  • Sérum conforme → 8100 si assez de matériel
  • Autres sérum → statif bleu
  • Autre type de tube → statif orange

🧬 Microbiologie moléculaire

  • Tous les tubes → statif violet

🦠 Bactériologie

  • Pochette + feuille → bac noir
  • Si tube LCR → avertir bactério par téléphone (heures d'ouverture uniquement)

📦 Envois

  • Frigo côté 8100 → statif gris
  • Congélateur près poste envois
  • Bac envois poste 1 enregistrements
